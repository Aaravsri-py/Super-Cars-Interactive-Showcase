# 🏎️ Supercars Store: Pure HTML/CSS Interactive Showcase

An immersive, futuristic web experience designed to showcase 15 legendary automotive machines. Built strictly around the constraints of **Zero JavaScript for layout and routing**, this project leverages cutting-edge CSS selectors to handle conditional logic, multi-page component routing, and an entry-lock gateway.

---

## 🧠 Core Engineering Highlights
* **CSS-Only State Machine:** Uses the modern CSS `:has()` pseudo-class paired with form state pseudo-selectors (`:valid`, `:invalid`) to build a functional site lock/security page entirely via stylesheet logic.
* **No-JS Routing Engine:** Utilizes CSS `:target` selectors to handle full-screen detail views and seamless overlay state transitions dynamically.
* **Cinematic Video Architecture:** Embedded HTML5 background rendering loops synchronized with advanced CSS animation keyframes (`curtain`, `revealUp`, `tilt`) to provide an automated, high-end showroom feel.
* **Complex Structural Animations:** Includes multi-element staggering (`nth-child` animation delays) and high-performance perspective shifts to make static grids feel exceptionally organic.

---

## 🛠️ Tech Stack
* **HTML5:** Advanced semantic layout structures, forms validation patterns, and native multi-source media streaming nodes.
* **CSS3:** Modern CSS features (`:has()`, `:target`, `backdrop-filter`, CSS Grid layouts, perspective 3D matrices).
